## 📘 About This Repository

This repository contains several real-world machine learning projects using **Scikit-Learn**.  
Each project includes:

✅ Data preprocessing  
✅ Feature engineering  
✅ Model training & evaluation  
✅ Performance metrics  
✅ Visualizations

These projects are **great for learning ML fundamentals**, building portfolio experience, and preparing for interviews.
